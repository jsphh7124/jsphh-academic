# jsphh-academic

Academic research skills for Claude Code, designed for OB/HRM researchers and professors.

## Installation

```bash
/plugin install github:jsphh/jsphh-academic
```

## Skills

| Skill | Description |
|-------|-------------|
| `academic-editor` | 學術論文英文編修 |
| `content-research-writer` | 研究內容寫作 |
| `literature-review` | 文獻回顧（含模板、資料庫策略、引用驗證腳本） |
| `scientific-critical-thinking` | 科學批判性思維（含邏輯謬誤、統計陷阱等參考資料） |
| `nlm-skill` | NLM 相關工具 |
| `social-psych-activity` | 社會心理學活動設計 |
